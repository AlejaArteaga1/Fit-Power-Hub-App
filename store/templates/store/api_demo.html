Django REST Framework API Demo

This page demonstrates the use of Django REST Framework to create a RESTful API for our fitness store.

Note: This API is built with Django REST Framework and follows REST conventions.
API Endpoints
  • GET /store/api/products/
    List all products (with pagination)
  • GET /store/api/products/{id}/
    Get specific product details
  • GET /store/api/categories/
    List all categories
  • GET /store/api/stats/
    Product statistics
  • GET /store/api/products/search/?q=protein
    Search products
  • GET /store/api/products/by_category/?category=SUP
    Filter by category

API Features
Filtering

Filter products by category, price range, and more using query parameters.

/api/products/?category=SUP&min_price=10&max_price=100
Search

Full-text search across product names and descriptions.

/api/products/search/?q=whey+protein
Sorting

Sort results by price, name, or creation date.

/api/products/?ordering=price
Pagination

Automatic pagination with 10 items per page.

/api/products/?page=2
Authentication

Protected endpoints for user-specific data like orders.

/api/orders/ (requires login)
Statistics

Analytics endpoints for business intelligence.

/api/stats/
