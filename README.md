# Fitpower Hub - Fitness Store & Nutrition Planner

## Description
Fitpower Hub is a Django web application combining an online fitness store with a nutrition planning system.

## Features
- Online store for supplements, clothing, and equipment
- Shopping cart with CRUD functionality
- User authentication system
- Nutrition planner with macro calculator
- Django REST Framework API
- Admin panel for product management

## Requirements
- Python 3.8+
- pip
